# SmartCart Customer Segmentation

## 📌 Overview
This project performs customer segmentation using clustering techniques to group customers based on their purchasing behavior and demographic information. The insights can help businesses improve targeted marketing and customer engagement.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## ⚙️ Workflow
- Data Preprocessing
- Missing Value Handling
- Feature Scaling
- PCA (Dimensionality Reduction)
- Agglomerative Clustering
- Cluster Visualization
- Customer Segment Analysis

## 📊 Techniques Used
- Principal Component Analysis (PCA)
- Agglomerative Clustering

## 📈 Results
- Customers were grouped into **4 distinct clusters** based on their characteristics.
- Cluster analysis was performed using spending patterns and income to identify customer segments.
